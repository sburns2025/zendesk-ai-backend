# Zendesk AI Backend

A secure Node.js proxy for Zendesk apps to call OpenAI without exposing your API key.

### 📦 Setup
```bash
npm install
```

### 🔐 Environment
Create a `.env` file:
```bash
OPENAI_API_KEY=sk-your-real-api-key
```

### 🚀 Run Locally
```bash
node server.js
```

Deploy with Vercel for best compatibility.
